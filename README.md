# Snobbin

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Drizzle

Create migration

```bash
npx drizzle-kit generate
```

Run migration

```bash
npx drizzle-kit migrate
```
